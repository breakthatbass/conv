# conv
Convert decimal, hexidecimal, and binary numbers

## usage

`conv [-d] [-x] [-b] [-h] <number>`

### flags
```
-d	decimal: convert dec to bin and hex
-x	hexidecimal: convert hex to dec and bin
-b	binary: convert bin to dec and hex
-h	show this help message and exit
```

## installation

```
git clone

cd conv

make install
```
